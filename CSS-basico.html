Definir estilos con HTML y CSS

CSS (cansade style sheets) se usa para controlar el formato visual de una pagina.

Con CSS se puede controlar el color,fuente,tamaño del texto,espaciado,alineamiento, imagen de fondo,como se ve una pagina dependiendo del tamaño de pantalla del dispositivo que estamos usando.etc.

Formas de usar el CSS

  • inlineusando el atributo style dentro del elemento HTML.
  • internal - usando la etiqueta <style> dentro de la seccion del <head>

    • inline CSS

    se usa para aplicar um estilo a un elemento HTML en concreto mediante el atributo style dentro de la etiqueta del elemento,ejemplo un titulo en color azul.

un titulo azul

Internal CSS

se usa para aplicar a un documento en concreto y se define dentro de la etiqueta <style> dentro de la seccion head del documento. ref en la linea 9 del codigo del documento.

External CSS

Es un archivo en extencion CSS que contiene exclusivamente codigo en CSS, se puede aplicar al documento mediante el elemento ><link>ref linea 8.

las dos principales ventajas de externalizar el css son:

  • los estilos se pueden aplicar a tantos documentos como queramos
  • la lomgitud del codigo disminuye sensiblemente
