La propiedad position

Esta propiedad define el metodo de posicionamiento de un elemento. los hay de 5 tipos.

  • static
  • relative
  • fixed
  • absolute
  • sticky

los elementos se posicionan con las propiedades top, left,right, bottom. Sin embargo estas propiedades no funcionaran sin especificar primero la position ,a demas, dependiendo del tipo de posicion las propiedades funcionaran de manera distinta.

position: static

todos los elementos tienen este valor por defecto

Las propiedades top,bottom, left y right no afectan a los elementos con posicion estatica.

position: relative

Un elemento con este valor se posicionara de manera relativa a su posicion normal, es decir, el control de ejersera sobre si mismo.

Este elenemto div tiene un position: relative

position: fixed

Un elemento con este valor, se posicionara relativamente respecto a la pantalla. lo que quiere decir que tendra siempre la misma posicion incluso si scrollemos la web.

position: absolud

Un elemento con este valor se posicionara respecto a su contenedor posicionado(no estatico)

si un elemento absoluto no tiene ancestros posicionados, usara el body y actura como un position: fixed

Este elemento div tiene un position:relative
Este elemento div tiene un position:absolud

position: stick

un elemeneto stick alternara su comportamiento entre relative y fixed dependiendo de la posicion de scroll de la pagina

El posicionamiento vertical con z-index

El z-index determina el orden en el que se amontonan los elementos.

Cuando los elementos se posicionan, puede que se monten sobre otros.

Esta propiedad determina en qué orden se amontonan los elementos. Es decir, qué elementos van por encima o debajo de los otros.
