Grid o Rejilla

El sistema de grid de css permite utilizar una plantilla de rejillas, con filas y columnas,apara alinear de manera sencilla paginas web sin neceidad de usar float o posicionamiento.

Elemento grid

Una rejulla consiste en un elemneto padre con 1 o mas hijos

El elemnto padre debe tener la propiedad display: grid o inlene-grid. los elementos decendientes se convertiran automaticamente en objetops(item) grid.
