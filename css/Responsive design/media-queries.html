Media queries

En ccs3, se introdujo la idea de que en vez de adaptar la web segun el tipo de dispositivo desde el que se esta visualiznado,las adaptaremos segun las capacidades.

Esto quiere decir que si en ves de revizar si en dispositivo es una pantalla, unh movil, una impresora..., simplemente se vasara enlos siguientes parametros:

  • ancho y alto del viewport
  • ancho y alto de la pantalla del dispositivo
  • orientacion- si el dispositivo esta en horizontal o vertical
  • resolucion

Usar media queries, nos capacita con la posivilidad de hacer paginas a medida para pantallas de escritorio:portatiles, tablets, moviles...

Sintaxis

Una expresion de media query puede contener una o varias condiciones que se resuelven en true o false 

            @media not | only mediatype and condiciones {
                codigo css
            }
