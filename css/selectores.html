Selectores CSS

los selectores tienen la funcion de encontrar o seleccionar elementos html a los que queremos dar estilo

  • selectores sencillos: Apuntan a elementos basandose en su clase(.), id(#) o nombre(h1)
  • Combinadores: selectores basados en una especifica relacion entre los elementos, ejemplo: .contenedor div
  • pseudo-clases: son selectores basados en el estado del elementos Eje:button:hover
  • pseudo elemnetos: Seleciona una parte especifica de un elemento
  • De atributo: Seleciona elementos basandose en un atributo o en el valorde un atributo

Selectores de nombre de elementos

selecciona los elementos basandose en su nombre de elemneto Ejemplo: todos los span seran de color morados 

            span {
                color: blueviolet;
            }

Selectores de id

Este selector usa el atributo id de un elemento html para selecionar un elemento espacifico

la id de un elemto es unica en el documentos.por lo tanto, el selecionador de id(#) se usa para especificar un unico elemneto

Ejemplo: Este parafo sera el unico en estar centrado y de color Azul.

El selector universal

El selector universal * seleciona a todos los elemntos.

Selector de agrupacion

sirve para agrupar aquellos selectores que tienen las mismas caracteristica

Ejemplo: 

            
            .sombras {
                box-shadow: 5px 5px 3px -2px;
            }    

         h1 {
            box-shadow: 5px 5px 3px -2px;
         }

         h2 {
            box-shadow: 5px 5px 3px -2px;
         }

.sombras, h1, h2 { box-shadow: 5px 5px 3px -2px; }
