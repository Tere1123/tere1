Atributos de elemnetos HTML

los atributos proveen informacion de elementos

  • Todos los elementos HTMLpueden tener atributos
  • Los atributos proveen info adicional de los elementos
  • los atributos se especifican en la etiqueta de apertura
  • Los atributos normalmente tienen estructura de nombre ="valor"

Ejemplo: Un link se define como la etiqueta <a>. El articulo "href" especifica la URL a la pagina a la que ira el link.
