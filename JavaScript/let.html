Todo sobre let

La palabra clave let tiene 3 usos principales:

  • Las variables declaradas con let no pueden ser redeclaradas
  • Las variables definidas con let deben declararse antes de ser usadas
  • las variables declaradas con let tiene un alcance (scope) de bloque

No se pueden redeclarar

En JS, al usar var se puede accidentalmente redeclaar y cambiar el valor de una variable

var x 'Juan' ;
var x = 0;

no solo se puerde el valor anterior si no que adicional genera errores

let impide redeclarar variables y por tanto es mas seguro.

Alcande de bloque (C)

Las variables declaradas con let tienen alcance de bloque, es decir, aquellas variables declaradas dentro de bloque {} no pueden ser usadas dentro del bloque Ref.linea 1 JS

Variables declaradas con let deben ser declaradas antes de usar

Usando var podemos usar variables mientras estas estan declaradas en cualquier parte del codigo
