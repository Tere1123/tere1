Objetos en javaScipt

Un objeto es un tipo espacial de variable, que almacena distintos valores clasificados como propiedades.

Un objeto tambien puede almacenar dististos metodos estos son bloques de codigo(funciones) que se ejecutaran en determinados momentos

En la vida real un coche es un objeto. un coche tiene propiedades como peso y color, y metodos como aparcar y parar.

Todos los coches tienen las mismas propiedades, pero estas tienen un valor dependiendo el coche, todos los coches tienen los mismos metodos pero estos se pueden ejecutar en distintos momentos

Normal mente usamos una variable para guardar un valor sencillo:

let coche = 'renault'

Los objetos son variables que puedan contenar multiple valores: 

        
     const coche = {
          marca:"Renaul",
          modelo:"Magane",
          color:"gris"
     }

Los valores se escriben emparejados con el nombre de la propieda name: value.
