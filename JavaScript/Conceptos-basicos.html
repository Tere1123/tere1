Conceptos JS

statemests o declaraciones

un programa informatico es una lista de instrucciones que se ejecuta por un ordenador.

En programacion estas instrucciones se llaman declaraciones o "statements"

un el programa de JS es una lista de declaraciones de programacion

las de claraciones se componen:

valores(values),operadores(operators),palabras claves(keywords) y comentarios(comments).
ejemplo;Tenemos una declaracion (statements) que le indica al navegador que debe escribir "Hola mundo" dentro de un elemento HTML con la id="test"

document.getElementById("test").innerHTML = "Hola mundo" ;

los statements o lineas de codigo ,se ejecutan en un orden de lectura

Punto y coma (semicolon);

se usan para separar los statements en JS, no son obligatorios pero si muy recomendados para buenas practicas de codigo. Ref al JS linea 1

Espacios en blanco

JS ignora los espacios en blanco ,y se usan para hacer el codigo mas legible, jemeplo: ref Js LINEA 11

let persona = "carlos";

Bloques de codigo

Las declaraciones o statements se pueden agrupar en bloques dentro de llaves(curly brackets) {...}

El proposito de esto es definir una serie de declaraciones para que se ejecuten juntas.

Un ejemplo de declaraciones unidas en bloques son las funciones. Ref JS linea 16

Palabras claves o keywords

A menudo muchas lineas de codigo inician por una palabra que indica la accion a realizar.
Esta palabras estan reservadas y no pueden usarse en ningun otro contexto.Ejemplo:

  • var - declara una variable
  • let - dechara una variable en bloque
  • const - declara una constante de bloque
  • if - indica un bloque de codigo a ejecutarse cuando se cumple una condicion
  • swich - indica un bloque de codigo a ejecutarse dependiendo de distintos factores
  • for - Indica un bloque de codigo a ejecutarse en bucle
  • function - Declara una funcion
  • return - Sale una funcion
  • try - Implementa manejo de errores a un bloque de codigo
