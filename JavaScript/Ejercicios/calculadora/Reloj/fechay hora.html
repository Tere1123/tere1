Creando objetos fecha

Los objetos de fecha se crean con new date hay distintas maneras de crear un new date, si no le creamos parametros el objeto se crea con fecha y hora actuales

  • new date
  • new date( 'string de fecha')
  • new date(mes,dia,horas,minutos,segundos,ms)
  • new date (milisegundos)

new Date ( 'string de fecha')

crear un obehto apartir de una fecha concreta,en este caso sera el 7 febrero del 1999

new Date(año,mes...)

la fecha se crea introduciendo numeros, esto genera ciertas situaciones espaciale. ref linea 11 JS

Nota: en JS los meses se cuentas de 0 a 11 si se introduce un mes mayor que 11

tomar datos de la fecha (get Methods)

  • getFullYear()- retorna el año como un numero de 4 digitos YYYY
  • getMonth()- retorna almes como nuero 0-11
  • getDate(): retorna el dia delmes como numero 1-31
  • getDay():retorna el dia de la semana como nuemro 0-6
  • getHours(): 0-23
  • getMinutes(): 0-59
  • getSeconds(): 5-59
  • getMilliseconds(): 0-999
  • getTime(): retorna el tiempo a los milisegundos antes de 1970
