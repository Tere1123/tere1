Valores de JS

hay dos tiposde valores: fijos y variables
A los fijos se les llama literalmente (Literals)
A los variables se les llama variable

Valores fijos,literales

Los dos tipos de Literales son los nuemros y los strings, o cadenas de caracteres.

Los numeros se escriben con o sin decimales

1001

strings son textos que pueden ir escritos entre comillas dobles o simples

'Jorge Gonzalez'

Identificadores/nombres

Los identificadores se usan para nombrar variables y funciones.
Hay una serie de reglas para nombrar comunes en la mayoria de lenguajes de programacion.
un identificador debe iniciar por:

  • una letra (A-Z o a-z)
  • un simbolo de dolar
  • o una barra baja _

Los caracteres siguientes pueden ser numeros, letras, _, $.

no se permite iniciar con numeros

Uso de mayusculas

Los identificadores son sensibles a las mayusculas. esto quiere decir que lastName y lastname son dos variables diferentes

por ejemplo, JS no interpreta LET o Let como ña palabra clave let

Camel Case

Antes los programadores utilizaban guiones para unir distintas palabras en una variable last-name, first-name...

Sin embargo los guiones estan reservados para las restas, por lo que los programadores de JS inician a separar las diferentes palabras mediante mayusculas dejando el resto en mayusculas
